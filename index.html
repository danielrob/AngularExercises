Angular Forms

1. <form name="AngularForm"> - this associates the form with an angular object which is seen below. You can change the submitted status. 


{{AngularForm | json }}

2. Add <input ng-model="something"> - here the difference is the input has an ng-model directive. Note that you can now make pristine change. 


{{AngularForm1 | json }}

3. Add < input name="john"> - here the difference is the input has a name. Now the form has an john attribute. 


{{AngularForm2 | json }}

4. Add < form ng-submit="myVar='intercepted!'"> - here the difference is the input has a name. Now you can access the json for AngularForm.name, which is shown below.

I have been: {{myVar}}

5. Using ng-click instead The same thing can be achieved using ng-click on the first input/button with type submit in the form. Don't use both.

I have been: {{tyd8o}}
