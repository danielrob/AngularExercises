ng-class=

With Booleans

"{green:true}" - I get the class green

"{green:false}" - I don't get the class green

With Expressions

ng-class={green:4==4}

ng-class={green:4==5}

With Strings

(Strings effectively become variable names)

ng-init="style='green'
ng-class="style"

ng-class="unkownstyle"

With Arrays of Strings

(Work like strings, but more than one possible class)

ng-init="style='green'"
ng-class="[style, style1]"

ng-init="style='green'; style2='fancy'"
ng-class="[style, style2]"

With Hashes

(Evaluates the value of the hash key to a 'truthy or falsey' value to determine class)

ng-class="{large:hashkey}
